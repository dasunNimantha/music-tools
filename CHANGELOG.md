## [0.1.0] - Initial Release

### ‚ú® Features

- Multi-utility portal with home screen
- **Metadata Editor**: Edit artist, album, genre, year, and cover art for music files
- **Music Downloader**: Placeholder for future music download functionality
- **Audio Converter**: Placeholder for future audio conversion functionality
- Dark/Light theme toggle with Cosmic-inspired UI
- File selection via file picker or folder scanning
- Batch processing of multiple audio files
- Real-time metadata preview
- Error logging with scrollable view
- Loading animations and visual feedback
- Support for MP3, FLAC, M4A, OGG, WMA, AAC, MP4, OPUS formats
